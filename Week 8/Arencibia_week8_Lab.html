The textbooks chapter on linear models (Line Up, Please) introduces linear predictive modeling using the workhorse tool known as multiple regression. The term multiple regression has an odd history, dating back to an early scientific observation of a phenomenon called regression to the mean. These days, multiple regression is just an interesting name for using a simple linear modeling technique to measuring the connection between one or more predictor variables and an outcome variable.

Step 6.4 - Analysis

Which regression model works best? Which of the predictors are statistically significant in each model? If you wanted to create the most parsimonious model (i.e., the one that did the best job with the fewest predictors), what would it contain? You MUST answer these questions.

  1. The linear model (also known and regression model) works when we have two or more variables to ascertain a relationship. This works best with large datasets as we are able to see what items correlate. But we must always remember that Correlation does not equal causation! lm() is best used when we have numerical values.

  2. Model 1 Winter Severity(X4) is statistically significant with a P Value of less than .05 (value=0.036263). Model 2 Adult population(X2) is statistically significant with a P Value of less than .05 (value=0.0112). Model 3 All values are statistically significant.

If we are going with the model with the most variance we would choose Model 1, since it is accounting for 54% of the Variance in the fawn population.The R-Squared is 54.6%.
